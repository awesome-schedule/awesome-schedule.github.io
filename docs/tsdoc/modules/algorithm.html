RawAlgoCourse

RawAlgoCourse: [string, number[], TimeArray, MeetingDate]
  • Defined in algorithm/ScheduleGenerator.ts:68

The data structure used in the algorithm to represent a Course that possibly has multiple sections combined (occurring at the same time)

0: key of this course 1: an array of section indices 2: TimeArray 3: Start and end date

Example:

["span20205", [0, 1, 2], [[600, 650, 1], [600, 650, 3], [], [], []], [8, 28, 12, 26]]
