add

  • add(schedule: RawAlgoSchedule): void
    • Defined in algorithm/ScheduleEvaluator.ts:357

    Add a schedule to the collection of results. Group the time blocks and sort them in order.

    remarks

    insertion sort is used as there are not many elements in each day array.

    remarks

    This method has a pretty high performance overhead. It will have a even higher overhead if the meeting room for each section is fetched and ordered, which is necessary to compute the total walking distance.
