Interface ScheduleAll<T>

represents all courses in a schedule, stored as (key, set of sections) pair

Note that if section is -1, it means that all sections are allowed. Otherwise, section should be a Set/array of object corresponding to each section

remarks

This field is called All (yes, with the first letter capitalized) due to historical reasons

Type parameters

  • T

    the type of the container used for the set of sections. By default, this is a set of numbers, corresponding to the id field of each section
